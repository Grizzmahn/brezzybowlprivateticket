# Breezy Bowl XX Private Ticket Sales

This project is a simple, static website for private ticket sales for the Breezy Bowl XX event. It allows users to register, view ticket prices, and purchase tickets using Bitcoin.

## Features

- **Event Information:** Displays available tours and ticket prices for Europe and North America.
- **User Registration:** Users can register with their details and set a password.
- **Ticket Purchase:** Registered users can purchase tickets (1–10 at a time) after entering the correct purchase password.
- **Bitcoin Payment:** Payment is accepted via Bitcoin, with a QR code and address provided.
- **Timer:** A 30-minute timer is started for the purchase process.
- **Responsive Design:** Basic responsive layout using CSS.

## Folder Structure

```
my project/
├── brezzybowlprivateticket.html
└── images/
    ├── logo.png
    ├── brown_background.jpg
    ├── red_jumping_background.jpg
    └── qr_code.png
```

## Usage

1. **Clone or download** this repository to your computer.
2. Place all required images in the `images` folder as shown above.
3. Open `brezzybowlprivateticket.html` in your web browser.

## Notes

- The purchase password is **BREEZYBOWL**.
- This site is for demonstration purposes only and does not process real payments.
- All form submissions and logic are handled client-side (in the browser).

## Customization

- To change ticket prices or event details, edit the HTML file directly.
- To update images, replace the files in the `images` folder.
